01 · The Recruiting Mindset
📈

Hire for Slope, Not Just Stats

Coachability and curiosity outrank a polished resume. The Readiness Review rewards reps who implement feedback fast — so we recruit for that trait on day one.

🎤

Always-On Talent Pipeline

Career panels, ERG panels, guest speakers and floor events double as a recruiting funnel — early-career talent sees the floor before they apply.

🌍

Diversity Fuels Innovation

An explicit commitment to equality, growth & empowerment. A diverse SDR floor builds a stronger bench for the entire GTM org.

🧭

Recruit Against the Bar

Because promotion criteria are public (the 5/5 review), candidates know exactly what "great" looks like before they accept the offer. No ambiguity, no surprises.

🤝

Manager-Owned Sourcing

Frontline managers carry a referral and screening role — the people who'll coach the rep help choose the rep.

🚀

Outbound-First Profile

As we rebalance to 55/45, we deliberately recruit for resilience and prospecting drive, not just inbound order-taking.

🔁 How the Agents Iterate the GTM

A closed loop — insight from one agent feeds the next, and the whole system gets sharper every cycle
1 · Build
  • Playbook agent sets the targets
  • Persona agent shapes the message
2 · Execute
  • Reps run sequences with AI personalization
  • Objection agent supports live deals
3 · Coach
  • Call Coach scores & develops reps
  • Onboarding agent closes readiness gaps
4 · Learn
  • Insights agent finds what's working
  • Findings update the playbook → repeat

Every agent keeps a human in the loop — managers review and approve before anything reaches a customer. This is the AI-first operating model I've already begun running, adapted for Avalara's reset.
